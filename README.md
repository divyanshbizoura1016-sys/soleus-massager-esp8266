# 🦵 Soleus Muscle Massager (ESP8266 Based)

The Soleus Muscle Massager is a compact biomedical device designed to stimulate the calf muscle using a combination of a solenoid valve and an air pump. It provides rhythmic compression to mimic natural muscle contractions, helping improve blood circulation, reduce fatigue, and relieve stiffness caused by prolonged inactivity.

## ⚙️ Working Principle
The system inflates an airbag for a fixed duration using an air pump and then releases the air through a solenoid valve. This cycle creates a compression–relaxation effect on the soleus muscle.

- Inflation time: Adjustable (default 10 seconds)  
- Deflation time: Fixed (15 seconds)

## ✨ Features
- Adjustable air filling duration using buttons  
- ON/OFF control switch  
- Automatic compression cycles  
- Portable and low-cost design  
- Based on ESP8266 (ESP-12F)

## 🛠️ Hardware Components
- ESP-12F (ESP8266)  
- Air pump  
- Solenoid valve  
- Relay module / MOSFET driver  
- Push buttons (increase/decrease)  
- ON/OFF switch  
- Power supply  

## 🔌 Pin Configuration
- Pump → D5  
- Solenoid → D6  
- Power Switch → D2  
- Increase Button → D7  
- Decrease Button → D8  

## 🚀 Applications
- Muscle recovery after exercise  
- Reducing stiffness due to long sitting hours  
- Improving lower limb blood circulation  
- Assistive physiotherapy device
